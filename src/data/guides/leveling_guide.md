# Guia para subir de nível

Para qualquer tipo de progressão de nível, ter um grupo completo (party) com personagens do mesmo nível é crucial. Em DMW, você ganha EXP bônus por ter um grupo completo.

Ao subir de nível, é altamente recomendável usar os itens/roupas listados abaixo para evoluir seus digis o mais rápido possível:
- **Uniforme amarelo de futebol para iniciantes**
- **Amplificador +1000 [Evento]**
- **Graça do Desafio [EXP] +1000%** (qualquer tipo de bônus de experiência de masmorras)
- **Conjunto completo de roupas de batalha futurista**

> **Observação Importante:** Quanto maior a diferença de nível entre os membros do grupo, menos experiência eles receberão. Há um limite de 7 níveis de diferença; quando houver 8 níveis de diferença entre os membros do grupo, eles não receberão experiência uns dos outros.
